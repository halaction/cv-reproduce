We reproduce the ConvMixer architecture of \citet{trockman2022patches} on CIFAR-10 and compare it against two compact reference baselines, ResNet18 \citep{he2016deep} and DeiT-Tiny \citep{touvron2021training}. The implementation uses a shared PyTorch training pipeline with \texttt{timm}-provided baselines, Weights \& Biases logging, and a \texttt{uv}-managed environment. In the current tracked runs, ConvMixer is the strongest model with best test accuracy $0.8848$, followed by ResNet18 with $0.8453$ and DeiT-Tiny with $0.6365$. At the same time, ConvMixer is slower per epoch than the other compared models in the present runs. We also report architectural and data-size ablations that help localize which ConvMixer components are responsible for the observed behavior.
