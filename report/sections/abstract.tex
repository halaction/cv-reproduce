We reproduce the ConvMixer architecture of \citet{trockman2022patches} on CIFAR-10 and compare it against two compact reference baselines, ResNet18 \citep{he2016deep} and DeiT-Tiny \citep{touvron2021training}. The implementation uses a shared PyTorch training pipeline with \texttt{timm}-provided baselines, Weights \& Biases logging, and a \texttt{uv}-managed environment. In the current tracked runs, ConvMixer is the strongest model by test accuracy, ResNet18 is second, and DeiT-Tiny is clearly weaker under the same overall reproduction setting. At the same time, ConvMixer is slower per epoch than the other compared models in the present runs. The report also reserves space for follow-up ablations targeting the ConvMixer patch stem, channel mixing block, and data-size robustness; those experiments are already represented in code but their results will be added after the remaining runs are complete.
